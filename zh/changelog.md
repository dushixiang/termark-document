# 更新日志

这里记录 Termark 的版本更新、问题修复和行为变更。

## v1.0.18

### 修复

- 修复 AI 执行命令超时的问题。
- 修复 AI 助手与终端抢夺焦点的问题。
- 修复主机密钥确认按钮的焦点处理。
- 修复深色模式下资产源选中状态不清晰的问题。
- 修复内置 Agent 工具调用与结果配对不一致的问题。
- 修复 Telnet 终端无输出，以及断线后可能频繁重连的问题。
- 修复 Agent 拒绝执行命令后导致上下文错误的问题。
- 修复执行命令可能卡住的问题。
- 修复应用锁显示前外观初始化不完整的问题。
- 修复切换语言可能导致页面崩溃的问题。

### 调整

- 移除 Claude Code ACP 适配。
- 优化 AI 设置。
- 优化 Agent 处理流程。
- 优化 SSH 认证失败提示。
- 编辑器改为异步加载。
- 优化终端与输入框的焦点处理。
