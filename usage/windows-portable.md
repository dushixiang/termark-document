# Windows Portable Mode

To make a portable Windows version of Termark, create a `data` folder in the same directory as the `.exe` binary.

## Directory Layout

```text
Termark/
|-- Termark.exe
\-- data/
```

## Steps

1. Prepare the Windows `.exe` binary for Termark.
2. Create a folder named `data` in the same directory as the `.exe`.
3. Copy or distribute the whole directory, or package it as a `.zip` file.

## Notes

- The `data` folder must stay at the same level as the `.exe`.
- The folder name must be `data`. Do not rename it.
